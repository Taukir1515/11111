# Using Git

## Checking existing branches
`git branch`

## Creating a new branch
`git branch branch_name`

## Entering a branch
`git checkout branch_name`

## Deleting a branch
- Can't delete current branch. So move to another branch before deletion.  


`git branch -d branch_name`
